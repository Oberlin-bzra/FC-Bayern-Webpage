# FC Bayern Webpage

## Beschreibung
Schulprojekt: Statische Webpage über den Fußballverein **FC Bayern München** mit Infos zu Geschichte, Spielern und Erfolgen.  

## Funktionen
- Vereinsinfos & Geschichte  
- Spielerübersicht  
- Titel & Erfolge  

## Technologien
- HTML5 & CSS3  
- Statische Webpage, läuft lokal im Browser
